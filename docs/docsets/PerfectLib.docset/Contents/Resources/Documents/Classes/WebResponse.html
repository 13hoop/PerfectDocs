WebResponse

public class WebResponse

Represents an outgoing web response. Handles the following tasks: - Collecting HTTP response headers & cookies. - Locating the response template file, parsing it, evaluating it and returning the resulting data. - Provides access to the WebRequest object.

  • pushChunked(_:)

    If HTTP chunked transfer encoding has not already been started, chunked starts and the existing HTTP headers are written. Any existing body data is sent to the client as a chunk and the body data is cleared for the next round of appendBody/appendBytes. Calls the given completion handler when done. The parameter to the handler will be true if the data was successfully pushed. If false is given then the request should be considered to have been aborted.
