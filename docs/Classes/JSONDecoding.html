JSONDecoding

public class JSONDecoding

This non-instantiable object serves as an access point to a registry for JSONConvertibleObjects A JSONConvertibleObject is a custom class or struct which can be converted to and from JSON.

  • JSONConvertibleObjectCreator

    Function which returns a new instance of a custom object which will have its members set based on the JSON data.
