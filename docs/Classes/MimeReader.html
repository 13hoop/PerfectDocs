MimeReader

public final class MimeReader

This class is responsible for reading multi-part POST form data, including handling file uploads. Data can be given for parsing in little bits at a time by calling the addTobuffer function. Any file uploads which are encountered will be written to the temporary directory indicated when the MimeReader is created. Temporary files will be deleted when this object is deinitialized.
